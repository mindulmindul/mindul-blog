## Linux内核的组成
Linux内核包含五大功能：
1. [进程管理（Process Management）](1.process_management/)
2. [内存管理（Memory Management）](2.memory_management/)
3. [虚拟文件系统（VFS，Virtual File System）](3.VFS/)
4. [设备驱动（Device Driver）](4.device_driver/)
5. [网络管理（Network）](5.net/)
